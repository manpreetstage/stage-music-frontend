✅ Supported Audio Formats

  • MP3 (Recommended)
  • WAV
  • M4A / AAC
  • FLAC
  • OGG
  • WMA

⚡ Quick Tips

  • Use high quality audio files (320kbps recommended)
  • Ensure file name is descriptive
  • Fill all metadata for better discovery
  • Lyricist credits give proper recognition
  • Song will be live immediately after upload!

📝 Required Fields

  • Song Title: Name of the song
  • Singer: Artist/Singer name
  • Audio File: The actual music file

🎨 Cover Image (Optional)

  • Upload album art or song poster
  • Recommended size: 1000x1000px or higher
  • Square format looks best
  • Supported: JPG, PNG, WEBP, GIF
  • If not uploaded, Stage logo will be used

🎯 Best Practices

  • Use clear, readable song titles
  • Include featured artists in singer field
  • Add lyricist name for proper credits
  • Mention movie name if it's from a film
